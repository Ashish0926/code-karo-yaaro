
# Code-Karo-Yaaro-ReactJs




## Run Locally

Clone the project

```bash
  git clone https://github.com/Ashish0926/code-karo-yaaro.git
```

Go to the project directory

```bash
  cd code-karo-yaaro
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

To deploy this project run

```bash
  npm run deploy
```
